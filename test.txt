git init: initializes git repository

cd .. : go back to parent folder

git status : Shows status of files in repository
git add . : Adds every file to the staging area
git commit -m <STRING> : Commits with message STRING
git log : log of all commits
git log --oneline : log of all commits, each in one line

Checkout Commit
	- Very safe
	- Read only
git checkout <commit ID>
git checkout master : goes back to latest version
	
Revert Commit
	- Undo a particular commit
	- Safe
git revert <commit ID>

Reset Commit
	- Goes back to a particular commit, deleting commits up to that point
	- Unsafe
git reset <commit ID> 
git reset <commit ID>  --hard : deletes actual changes as well

Branches
git branch <NAME> : Creates a new branch with name NAME
git branch -a : shows all the branches
git checkout <NAME> : starts saving changes in branch NAME
git branch -D <NAME> : deletes branch NAME 
git checkout -b <NAME> : creates a new branch NAME and switches to it

Merging
git merge <NAME> : merges NAME brach into master branch

Merging conflict
	1. Edit changes on test editor
	2. Stage and commit new master file

Github
git push <LINK> <NAME> : pushes branch NAME to Github repository in LINK
